Procedure

  1. Select ring number to measure
  2. Move scope to left tangent
  3. Select "Left" → Record x₁
  4. Move to right tangent
  5. Select "Right" → Record x₂
  6. Repeat for all 10 rings
  7. Go to Observation Table
